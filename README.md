# 人物志转换

把人物志问卷表格文件转换成易于编辑的文字段落

如果你不知道这个项目是干嘛的说明你大概率不需要它

# 使用方法

## 离线版

[releases](https://github.com/hydawn/renwuzhi/releases/latest)的Assets里下载最新版html文件到本地，用浏览器打开。然后跟着文字指示走

## 在线版

[https://hikerjoy.duckdns.org/renwuzhi/](https://hikerjoy.duckdns.org/renwuzhi/)

不推荐使用在线版，因为我忘了维护就会挂

# 开发

`make version`将项目编译为单文件`dist/renwuzhi_contained.html`和`dist/renwuzhi_contained.vx.x.x.html`

版本信息来源于`renwuzhi.html`的`<title>`里
